# TrendScout

AI-powered keyword research tool for SEO and content creators.

## Features

- 🔍 Smart keyword search
- 📊 Google Trends integration
- 🤖 AI-powered scoring
- 💰 Affordable pricing ($9-19/month)

## Tech Stack

- Frontend: Next.js 14 + Tailwind CSS
- Backend: FastAPI + PostgreSQL
- Deployment: Vercel + Railway
